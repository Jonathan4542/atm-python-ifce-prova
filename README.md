Este projeto é um sistema de caixa eletrônico desenvolvido em Python, com o objetivo de praticar conceitos de Programação Orientada a Objetos (POO).

O sistema permite criar contas bancárias (corrente e poupança) e realizar operações básicas, como depósito, saque, consulta de saldo e visualização de histórico.

A estrutura do projeto foi organizada em pastas separadas para facilitar a organização do código, dividindo responsabilidades como clientes, contas, operações e banco.

Além disso, foram aplicados conceitos importantes de POO, como herança, polimorfismo, encapsulamento, composição e agregação, tornando o código mais organizado e reutilizável.

O programa é executado pelo terminal, onde o usuário interage por meio de um menu simples para acessar as funcionalidades disponíveis.
